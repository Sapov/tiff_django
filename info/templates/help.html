Регистрация на сайте:

Для регистрации на сайте нужно перейти по адресу https://order.san-cd.ru/register/

  1. В форме ввести Имя, электронную почту и придумать пароль.
  2. На электронную почту прейдет письмо с ссылкой об активации аккаунта.
  3. После подтверждения регистрации вы попадаете на главную страницу сайта.

Заполнение профиля:

  1. Первым делом после регистрации нужно заполнить профиль пользователя:
  2. Перейдите в настройки / профиль пользователя
  3. Заполните профиль
  4. Укажите Имя, Отчество, Фамилию.
  5. Введите номер телефона
  6. Укажите если нужно (присылать уведомления на whatsapp)

Добавьте организацию плательщика:

  1. Перейдите в настройки / организации
  2. На странице "Мои организации" нажмите кнопку "Добавить организцию"
  3. Введите название или ИНН организации остальные поля заполнятся автоматически
  4. Введите БИК банка и введите расчетный счет организации. Все остальное заполниться автоматически.
  5. Нажмите сохранить

Как сделать заказ:

Загрузка файлов:

  1. Для создания заказа вам нужно загрузить файлы для печати. (Файл должен соответствовать требованиям)
  2. При загрузке требуется указать (количество, материал, финишную обработку (если есть).
  3. После загрузки - файл окажется в списке загруженных файлов. Где отобразяться его характеристики: материал, размер, финишная обработка, стоимость.
  4. Если что то при загрузке указали не верно (материал, размер, финишную обработку или колличество) - это можно поправить нажав кнопку изменить.
  5. Файл можно так же удалить нажав соответствующую кнопку.

Создание заказа:

  1. Для создания заказа нужно нажать кнопку: "Новый заказ"
  2. Откроется новая страница в которой нужно выбрать параметры заказа:
    • Выбрать организацию плательщик;
    • Выбрать метод доставки;
    • Выбрать дату готовности заказа;
    • Нажать на кнопку "Новый заказ".
  3. В следующем окне сформируется заказ. Снизу будут видны файлы которые можно добавить в заказ. Выберите файл и нажмите "Добавить в заказ".
  4. После добавления нужных файлов в заказ станет активна кнопка "Оформить заказ"
  5. Нажмите кнопку "Оформить заказ"
  6. После нажатия кнопки оформить заказ станет доступна страницы оплаты.

Удаление файлов из заказа

Для удаления файлов из заказа нужно нажать на файле в который добавлен в заказ кнопку удалить. Файл будет удален из заказа, но останется в базе и его можно добавить повторно или в другой заказ

Для удаления файла полностью его следует удалить из раздела "Мои файлы"

Оплата заказа:

Для оплаты заказа по безналичному расчету требуется указать организацию-плательщика для выставления счетов.

В сервисе реализованы два вида оплаты:

  1. Безналичный расчет с выставлением счета и получением закрывающих документов.
  2. Оплата картой через платежный шлюз с предоставлением чеков отчетности в налоговой.

На странице оплаты появятся оба варианта если если вы указали в заказе организацию плательщика. Если нет то будет доступна оплата только по карте.

Оплата по карте:

Если вам требуется максимальная оперативность - это самый быстрый способ запустить заказ в работу.

  • При нажатии кнопки "Оплата по карте"
  • Откроется страница платежного шлюза.
  • После оплаты на указанную вами почту придут "Чеки для отчета перед налоговой".
  • Статус заказ измениться на "Оплачен".

Заказ поступает в работу. Через некоторое время статус заказа измениться на статус "В работе"

Оплата по счету:

Для оплаты по счету при формировании заказа нужно выбрать плательщика из ваших организаций. На странице оплаты будет доступна кнопка "Получить счет". Счет будет выписан на ту организацию которую вы выбрали в заказе. После оплаты счета, оплата в системе появится автоматически и заказ изменит статус на "Оплачен".

Заказ поступает в работу. Через некоторое время статус заказа измениться на статус "В работе"

Добавление организации плательщика

  1. Перейдите в настройки и выберите Организации.
  2. На странице нажмите кнопку "Добавить организацию"
  3. Введите реквизиты организации.
  4. Нажмите сохранить.

Готовность заказа:

  1. По готовности заказа вам поступит сообщение на почту или Whatsapp о готовности.
  2. Закрывающие документы появятся на странице оплаты.
  3. Время и адрес получения заказы придут в уведомлении.

Доставка:

По умолчанию доставка отключена (в поле выбора доставки стоит "Самовывоз")

Сроки исполнения заказа

Сроки исполнения зависят от объема и сложности заказа. Обычно мы исполняем заказ за 2 дня. Если по каким либо причинам исполнение заказа затягивается, мы обязательно вас об этом ставим в известность.
